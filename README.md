# AWS-Vibe-Apps

Static vibe apps published with GitHub Pages.

## Apps

- [Chess Demo 1](./chess-demo1/) - Playable chess demo.
- [Chess Demo 2](./chess-demo2/) - Chess.com player stats dashboard.
- [AWS Academy Tracker](./aws-academy-tracker/dist/) - React tracker for AWS Academy progress.

## GitHub Pages

Publish this repository from `main` using the `/ (root)` folder.

Once Pages is enabled, the apps will be available at:

- `https://pabloin.github.io/AWS-Vibe-Apps/`
- `https://pabloin.github.io/AWS-Vibe-Apps/chess-demo1/`
- `https://pabloin.github.io/AWS-Vibe-Apps/chess-demo2/`
- `https://pabloin.github.io/AWS-Vibe-Apps/aws-academy-tracker/dist/`
